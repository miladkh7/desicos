The desicos.abaqus module includes the DESICOS plug-in for Abaqus whose functionalities can be used in two main ways, using the Graphic User Interface (GUI) or using the Python API.

  • GUI
    • Loading and configuring a model
    • Applying imperfections
    • Running the analysis
    • Post processing
  • Python API
    • Creating a Model from the Database
    • Applying Measured Imperfections using the Imperfection Database
    • Applying Measured Imperfections using Fourier Series
